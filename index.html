Mock Start-up

The idea for this project was to replicate a start-up website for a mobile application. This app would be used for comparing the best athletes in a chosen league. Two players would be presented and the user would then have to decide which player is better. The webpage uses HTML, CSS and Bootstrap as the library for this project. A challenged faced was allowing the webpage to be responsive on mobile considering it has a navbar and columns made with Bootstrap. Another challenge was allowing the phone image to lay under the next sections background colour.
